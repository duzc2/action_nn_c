# Version 1 开发任务清单

## 1. 文档目标

把 Version 1 拆成可以逐项完成、逐项打勾的任务清单。

Version 1 只做：

> **区域识别**

所以任务只覆盖：

- 数据采集
- 标签记录
- 数据集构建
- 数据报告
- generate / train / infer
- 测试与结果记录

## 2. 使用方式

建议按顺序执行：

1. 先完成前置配置任务
2. 再做工具链任务
3. 再做数据集任务
4. 再做训练推理任务
5. 最后做测试与结项

原则：

- 前一组任务没完成，不进下一组
- 每项任务完成后都要有产物

## 3. 任务状态建议

建议每项任务使用以下状态之一：

- `[ ]` 未开始
- `[-]` 进行中
- `[x]` 已完成
